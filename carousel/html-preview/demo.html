PiCarousel Playground Showcase

Slider 1: Basic autoplay, no infinite loop

Options: autoplay: true, infinite: false, direction: ltr, pauseOnHover: false
Plays automatically but stops at the last slide.

Slider 2: Infinite loop + autoplay

Options: autoplay: true, infinite: true, direction: ltr
Loops seamlessly back to the first slide and keeps playing.

Slider 3: Infinite autoplay (LTR)

Options: autoplay: true, infinite: true, direction: ltr
Same as Slider 2 but different images.

Slider 4: Manual control only (autoplay disabled)

Options: autoplay: false, infinite: false, direction: ltr
User must click buttons or dots. No auto movement.

Slider 5: RTL direction, infinite autoplay

Options: autoplay: true, infinite: true, direction: rtl
Scrolls from right to left, looping forever.

Slider 6: RTL direction, no autoplay

Options: autoplay: false, infinite: false, direction: rtl
Manual control, right-to-left, stops at ends.
